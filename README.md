# Aman Gourd - Digital Marketing Portfolio

A modern, responsive portfolio website for Aman Gourd, a Digital Marketing Manager with 10+ years of experience.

## Features

- **Responsive Design**: Fully responsive and mobile-friendly
- **Modern UI**: Clean, professional design with gradient accents
- **Interactive Elements**: Smooth scrolling, animations, and hover effects
- **Contact Form**: Functional contact form with validation
- **Timeline Layout**: Professional experience displayed in an interactive timeline
- **Project Showcase**: Grid layout for projects and achievements
- **Social Media Integration**: Links to professional profiles

## Sections

1. **Hero Section** - Introduction with call-to-action
2. **About Me** - Professional summary, stats, and core expertise
3. **Work Experience** - Timeline of professional experience
4. **Projects** - Featured projects and YouTube channel
5. **Contact** - Contact form and information

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)
